# 關於程式設計班選修課程

## 課程介紹
近年來由於108新課綱，越來越多學員積極投入資訊科技領域學習，學習的方向也漸漸從檢定與競賽變得更多樣化，舉凡網頁、資安、大語言模型、遊戲開發皆為熱門領域及選項。因應此趨勢，程式設計班於113學年度進階班進行專題課程試行，並於114學年度起正式新增課程，每學年開設課程可能略有不同。

| 年份             | 選修I        | 選修II             | 課程成果 |
|:---------------- |:------------ |:------------------ |:-------- |
| 113學年度 (試行) | 神經網路   |                    |          |
| 114學年度        | 個人網頁設計 | 競賽程式設計與技巧 | 敬請期待 |

## 個人網頁設計
### 簡介
本堂課程將引領同學從認識網頁架構開始，一步步教導學員基本的網頁設計知識及製作可互動的個人網頁。

招生人數上限: 40人。

### 選修需求 
無，建議同學應具備基本的文字型程式設計觀念(如Python、C++等)

### 教學團隊
主教學: 黃柏諺(程式設計班官網設計者)

副教學: 蘇丞泓(具有多種網頁前後端設計製作經驗)

助教: 李萱雅、張凱富、薛宇博

### 課程安排

| 節次 | 內容                                                     |
|:---- |:-------------------------------------------------------- |
| 1    | GitHub 帳號申請、基礎html+CSS認識、git使用               |
| 2    | npm安裝、Vite安裝 (React)、CSS選取器理解、TypeScript簡介 |
| 3    | 延續上周TypeScript練習、CSS美化                          |
| 4    | 產出個人介紹(Hexo模板)、 上傳到GitHub                     |

## 競賽程式設計與技巧
### 簡介 
本堂課程將在進階班上學期原有的動態規劃(DP)進行加深加廣，學習競賽中(如學科能力競賽、TOI等)常使用的進階DP知識與實作，並傳授競賽獨門技巧，幫助學員在競賽中百尺竿頭，更近一步!

招生人數上限: 50人。
### 選修需求
學員應具備演算法知識，並對基本DP有一定的熟悉程度(如: 背包問題)，想進一步往程式競賽精進者得選修此門課程。若初階班學員選修此門課程，需先 **完成一份作業以確保其具備基本的程式能力及對DP的基本理解**。
### 教學團隊
教學: 黃禹程(APCS 4+4，曾參與 2024年成大邀請賽(全國第八)、少年圖靈計畫複賽等多種程式競賽)(本人表示: 我把自己的APCS五級分送掉了)

助教: 黃彥凱、楊珈瑜

| 節次 | 內容           |
|:---- |:-------------- |
| 1    | 資料結構       |
| 2    | 進階DP(上)     |
| 3    | 進階DP(下)     |
| 4    | 競賽和實作技巧 |
