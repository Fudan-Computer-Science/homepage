入班考試資訊

114學年度插班考熱烈報名中​

插班考報名表單​

初階入班考​

  • 考試時間: 開學第一周的某個中午，12:20-12:50，共30分鐘。
  • 考試題型:
    • 觀念題 (20題，C++)
  • 範圍:
    • 基礎輸入輸出
    • 運算子
    • 判斷
    • 迴圈
    • 一維陣列
  • 講義連結:點我
  • 練習題: 第十二屆 題單

進階入班考(二次篩選)​

  • 考試時間: 2025 8/7 00:00 到 9/10 23:59，在截止前可無限次提交，不罰時

  • 考試題型:

    • 實作題 (C++)

  • 範圍:

    • 基礎語法 (輸入輸出、運算子、判斷、迴圈、陣列、字串)
    • STL Container
    • 指標 (Pointer)
    • DP
    • 矩陣快速冪
    • BFS / DFS

  • 考試系統: Dandanjudge 新用戶請註冊，暱稱設「班級 座號 姓名」

  • 報名流程: 1. 填寫報名表單，

  • 考古題:

    • 第11屆: 題目 題解

初階插班考​

  • 報名時間: 12/18 - 1/30
  • 考試時間: 1/1 - 2/22 Dandanjudge
  • 考試題型:
    • 實作題 (C++)
  • 範圍:
    • 上學期初階班上課內容，可參考第十四屆初階班上課講義及課程影片
  • 錄取人數: 5人

進階插班考​

  • 報名時間: 12/18 - 1/30
  • 考試時間: 1/1 - 2/22 Dandanjudge
  • 考試題型:
    • 實作題 (C++)
  • 範圍:
    • APCS中高級程度 (包含貪心、基本動態規劃、STL、排序、DFS、BFS等)，可參考第十四屆進階班上課講義及課程影片
  • 錄取人數: 1人
