Chapter 2.

Variables and Data Types

2-0 變數的命名​

命名一個變數，需要注意下列事項：

  1. 使用識別字
  2. 字母的大小寫是不同的且須半形字
  3. 數字不得為開頭且符號只能用**__**(底線)
  4. 中間不得有空格

※ 識別字與關鍵字的差別

  • 識別字：根據程式需求自行定義的名稱
  • 關鍵字：程式語言保留的關鍵字，作為指令與特殊用途

定義識別字時不可與關鍵字相符

2-2 變數的宣告​

  • 在日常生活中，我們會運用如x,y之類的代數帶入數字以簡化數字的運算，在程式中也有諸如此類的概念
  • 程式語言中，我們將數字、字元等資料存放在一個代替的名稱中，使程式設計師自由的運用，因此我們透過宣告變數的動作來讓每個名稱有意義
  • 未宣告變數前電腦是不認識他的，所以變數在使用前都必須要先宣告
  • 在宣告名稱前，我們會加一個存放的資料型態(下例為int、char)來定義他，電腦中的每筆資料都由數字構成，而資料型態就是這個數字的範圍，若超過這個範圍就會得到亂碼，稱為溢位(overflow)。
  • 在定義資料型態為字元的變數時，須以雙單引號夾字元作定義(eg：'a')。
  • 可用”,”(逗號)做連續宣告(eg：若要宣告整數變數a及b，可將 int a;int b; 合併為 int a,b; )

重要觀念釐清​

  • int b=5; 的涵義是將 5(右)設定給 b(左)，不同於數學常用的等於，此處一個等於是“將…定義為…”的意思。
  • 而在數學裡的”=”，在程式裡我們用”==”來表示，同學需特別注意。

2-4 基本的輸入​

方法：使用 cin 與>>的搭配

  • 例：
cin>>a; //將輸入的東西儲存進變數 a 裡面

若要連續輸入的話，可以用多個>>來完成

  • 例：
cin>>a>>b; //連續輸入變數 a 跟 b

※易錯重點：
輸入多變數時不需打空格，例：cin>>a>>””>>b;
或是在結尾作換行，例：cin>>a>>b>>endl ;

小練習​

請參考以下程式碼，並請問以下哪一組輸入可讓程式輸出錯誤答案？
#include <iostream>
using namespace std;
int main()
{
	int a, b;
	cin >> a >> b;
	cout << a + b << endl;
	return 0;
}

(A) 100 1000
(B) 1000000000 1000000000
(C) 2147483647 1
(D) -2147483647 -1

答案

(C)

(題源: 108學年度入班考)

對於以下虛擬碼，試問輸出為何?

(A) 8
(B) 16
(C) 24
(D) 32

int main(){
    int a = 32, b = 24, tmp;
    while (b != 0)
    {
        tmp = a % b;
        a = b;
        b = tmp;
    }
    cout << a << endl;
}
答案

(A)

(題源: 109學年度入班考)

對於以下概念，下列何者正確?

(A) int型態的數字限制範圍大於double
(B) int是32位元有號整數
(C) float的精度大於double
(D) bool是8位元有號整數

答案

(B)

(題源: 109學年度入班考)

旦旦監獄某次段考全校平均太低，為了調整分數老師們協議將每個人的分數加7分後乘以三再減五最後除以2 (不考慮分數大於 100 的狀況)，為了讓分數換更快速，某位老師寫了以下程式，請問程式碼中空白處應填入何者才可出目標分數?

(A) fscore = (iscore + 7) * (3 - 5 / 2);
(B) fscore = (iscore + 7) * 3 - 5 / 2;
(C) fscore = [(iscore + 7) * 3 - 5] / 2;
(D) fscore = ((iscore + 7) * 3 - 5) / 2;

#include<iostream>
using namespace std;
int main(){
    int iscore, fscore;
    cin >> iscore;
    /*空白處*/
    cout << fscore << endl;
}
答案

(D)

(題源: 109學年度入班考)

對於以下程式碼，試問輸出為何?

(A) 1
(B) 0
(C) 無法輸出
(D) 以上皆非

#include<iostream>
using namespace std;
int main(){
    bool t = true, f = false;
    cout << (!t  f && t  t && t) << endl;
}
答案

(A)

(題源: 109學年度入班考)
