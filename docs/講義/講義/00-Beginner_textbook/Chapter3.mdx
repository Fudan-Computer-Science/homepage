## 3-1 常見的運算子

### 運算子
運算子是一種記號，讓程式知道要做什麼，以下是常見的計算子：

    - #### 設定運算子 \= 
        Def. 將左方**設定**為右方數值  
        例：a=b;   //將b的值設定給a(並非a等於b)

    - #### 括號運算子 ()
        Def. 決定運算的先後順序   
        例：a-(b+c);  //(b+c)要先算

    - #### 算數運算子
        | \+ | \- | \* | / | % |
        | :---: | :---: | :---: | :---: | :---: |
        | 加法 | 減法 | 乘法 | 除法 | 取左邊除以右邊後的餘數 |

    - #### 複合賦值運算子
        | 類型 | a+=b | a-=b | a\*=b | a/=b | a%=b |
        | :---- | :---- | :---- | :---- | :---- | :---- |
        | 定義 | a的值設定為a+b | a的值設定為a-b | a的值設定為a\*b | a的值設定為a/b | a的值設定為a%b |
        | 等價於 | a=a+b | a=a-b | a=a\*b | a=a/b | a=a%b |

    - #### 遞增運算子
        |  | a遞增 | a遞減 |
        | :---- | :---- | :---- |
        | 可寫為 | a++//a=a+1或a+=1 | a--//a=a-1或a-=1 |

    - #### 關係運算子
        | \> | \< | \>= | \<= | \== | \!= |
        | :---: | :---: | :---: | :---: | :---: | :---: |
        | 大於 | 小於 | 大於等於 | 小於等於 | 等於 | 不等於 |

    - #### 邏輯運算子
        | && | \|\| |
        | :---: | :---: |
        | and 且 **(都要成立)** | or 或 **(其中一個成立)** |

    - #### 一元運算子
        | \+ | \- | \! | \~ |
        | :---: | :---: | :---: | :---: |
        | 正數 | 負數 | not 否 | 取1的補數 |  

        優先順序：\!(not) \> &&(and) \> \| \|(or)

    - #### 三元條件運算子
        `(條件) ? A : B  `
        如果條件成立，則A  
        不成立，則B

## 3-2 判斷  

### if判斷

    1. `if(/*條件式*/) {/*成立時要執行的程式*/}` ：  
        - 當小括號內的判斷條件成立後  
        就會執行大括號內的程式  

    2. else if：
        - 可有多個else if  
            - 前面需先有if或else if假設，當前面的假設不成立時才會執行else if判斷

    3. else：
        - 當上方if和else if皆不成立後才會執行  
        - 只能有一個else
        - else不需判斷

## 小練習
- 我們都知道邪惡的108年會考已經結束了，但程設班變態的初階教學用特殊管道取得了幾位學弟妹的國文科答對題數並試圖寫一支程式來得知大家的等級標示，而以下為他的程式以及108年會考國文科的答對題數及等級標示換算，請問他輸入題數為多少時程式輸出的等級會與對照表不一樣？
| 級別 | 答對題數 |
| ---- |:-------- |
| A++  | 45-48    |
| A+   | 43-44    |
| A    | 41-42    |
| B++  | 36-40    |
| B+   | 31-35    |
| B    | 19-30    |
| C    | 0-18     |

 (A) 16 
 (B) 43  
 (C) 25 
 (D) 42
 ```cpp
#include <iostream>
using namespace std;
int main()
{
	int num;
	cin >> num;
	if (num >= 45)
	{
		cout << “A++” << endl;
	}
	else if (num >= 43)
	{
		cout << “A+” << endl;
	}
	else if (num >= 36)
	{
		cout << “B++” << endl;
	}
	else if (num >= 41)
	{
		cout << “A” << endl;
	}
	else if (num >= 19)
	{
		cout << “B” << endl;
	}
	else if (num >= 31)
	{
		cout << “B+” << endl;
	}
	else
	{
		cout << “C” << endl;
	}
	return 0;
}
```
(題源: 108學年度入班考)

- 請問程式碼 `int a;` 所代表的含意是? 
(A)	執行函數a所包含的內容  
(B)	將程式碼存取至a的記憶體位址，並將a設為該程式碼之指標 
(C)	宣告a為一變數，且a可為任一數字  
(D)	宣告a為一整數變數，且a的值受到限制
