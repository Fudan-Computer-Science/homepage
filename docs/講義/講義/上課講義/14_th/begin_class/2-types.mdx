### 變數宣告
#### 宣告語法
在要宣告變數時，格式為`資料型態 變數名稱`，最後再根據情況是否要給初始值。

#### 正確的宣告
```cpp=
#include<iostream>
using namespace std;
int main(){
    double a,b;  //不同的變數之間以逗號隔開
    
    long long A; //大小寫是不一樣的變數喔OwO
    
    int d=0,e=1; //在宣告的時候可以給予初始值
}
```
#### 一些錯誤的宣告方式
```cpp=
#include<iostream>
using namespace std;
int main(){
    int double; //變數名稱不能是保留字，像是 int , return 這種的都不行; 
    
    int ａ; //不能使用全形字當作名稱，如果不小心用到請 shift+空白鍵
    
    int 1peko;  //數字不能當作字首
    int i love nakiriAyame; //變數名稱之間不能有空格
}
```
很多符號也不能使用，這裡就不一一列舉了，通常我們也只會用英為字母來當作變數名稱，有時會在同樣名稱的變數後方家數字以區別

:::warning
**如果沒有賦值，就做運算或輸出,很有可能會出錯!!像是:**
```cpp=
#include<iostream>
using namespace std;
int main(){
    int c;
    c++;
    cout<<c; //不一定會輸出1
}
```
雖然在很多編譯器都會預設為0，不過judge不一定會預設為0，所以會造成測試輸出是對的，放到judge卻是錯的
:::

#### 全域變數
就是宣告在`int main()`之上的變數，在程式編譯時就被宣告好(宣告在`int main()`裡面的稱為區域變數，在執行時才會宣告)，其他使用都和區域變數一樣，範例:
```cpp=
#include<iostream>
using namespace std;
int a;
int main(){
    cout<<a; //輸出0
}
```

全域變數有幾個特性，例如不須被初始化就會自己初始化好(整數型態初始化成`0`、小數型態初始化成`0.0`、字元型態初始化成`'\0'`(空字元)、邏輯型態初始化成`false`)

### 運算子

| 設定運算子 | 功能                | 示範      |
| --- | ----------------- | ------- |
| =   | 設定(賦值)，將右邊的值存到左邊的變數內 | `a = 1` |


| 單元運算子 | 功能             | 示範          |
| --- | -------------- | ----------- |
| +   | 正號             | `+a`        |
| -   | 負號             | `-a`        |
| ++  | 遞增 (+1) | `++a , a++` |
| --  | 遞減 (-1) | `--a , a--` |
| !   | 邏輯否    | `!a`        |
| \~  | 位元 NOT (二進位取反) | `~a`        |

看不懂```~```在幹嘛的話,其實就只是這樣(~a 會把 a 的每一個二進位位元逐位取反：0 → 1、1 → 0):
```
~a  ==  -a - 1
```


| 算數運算子 | 功能          | 示範      |
| --- | ----------- | ------- |
| +   | 加           | `a + b` |
| -   | 減           | `a - b` |
| \*  | 乘           | `a * b` |
| /   | 除 (整數除會捨小數) | `a / b` |
| %   | 餘數 (mod)    | `a % b` |

| 關係運算子 | 功能   | 示範       |
| --- | ---- | -------- |
| >   | 大於   | `a > b`  |
| \<   | 小於   | `a < b`  |
| >=  | 大於等於 | `a >= b` |
| \<=  | 小於等於 | `a <= b` |
| ==  | 等於   | `a == b` |
| !=  | 不等於  | `a != b` |


| 邏輯運算子  | 功能                   | 示範         |
| ---- | -------------------- | ---------- |
| &&   | 邏輯 AND (且)           | `a && b`   |
| \|\| | 邏輯 OR (或)            | `a \|\| b` |
| ^    | XOR (布林相異為真 / 位元互斥或) | `a ^ b`    |

| 運算子 | 功能        | 示範       |
| --- | --------- | -------- |
| &   |  AND    | `a & b`  |
| \|  |  OR     | `a \| b` |
| ^   |  XOR    | `a ^ b`  |
| \~  |  NOT    | `~a`     |
| \<\<  |  (乘 2ⁿ) | `a << n` |
| >>  |  (除 2ⁿ) | `a >> n` |

| 運算子 | 範例 (等同於)                 |
| --- | ------------------------ |
| +=  | `a += b` ⇔ `a = a + b`   |
| -=  | `a -= b` ⇔ `a = a - b`   |
| \*= | `a *= b` ⇔ `a = a * b`   |
| /=  | `a /= b` ⇔ `a = a / b`   |
| %=  | `a %= b` ⇔ `a = a % b`   |
| &=  | `a &= b` ⇔ `a = a & b`   |
| \|= | `a \|= b` ⇔ `a = a \| b` |
| ^=  | `a ^= b` ⇔ `a = a ^ b`   |
| \<\<= | `a <<= b` ⇔ `a = a << b` |
| >>= | `a >>= b` ⇔ `a = a >> b` |

#### 順序(上到下):
```
() [] . -> ++ --                 // 函數呼叫、陣列取值、成員存取、遞增遞減 (後置)
+ - ! ~ ++ -- (type) * & sizeof  // 單元運算子、型別轉換、取址、解參考
* / %                            // 乘法、除法、取餘數
+ -                              // 加法、減法
<< >>                            // 位移運算 (a >> n = a / 2^n)
< <= > >=                        // 關係運算子
== !=                            // 相等性運算子
&                                // 位元 AND
^                                // 位元 XOR
|                                // 位元 OR
&&                               // 邏輯 AND
||                               // 邏輯 OR
?:                               // 三元運算子
= += -= *= /= %= &= |= ^= <<= >>= // 指派運算子 (由右至左)
,                                // 逗號運算子

= += -= *= /= %= &= |= ^= <<= >>= (右至左)
```
雖然這樣說了 但還是一律建議在每一個運算子上加入```()```把要算的東西給括號起來 這樣不只可以讓別人看得懂 自己的思緒也會更清楚!!!
#### 注意!
1. 注意資料型態不同的變數不能做運算
2. 做題目記得，題目給的範圍不能超過你要用的資料型態範圍
3. 邏輯運算子要熟記結果

### 標準程式庫標頭檔
標頭檔決定了你可以使用哪些函式、類別、物件,引入不同的函式庫,能讓你的程式碼有不同的用途

例如:
```cpp=
#include <iostream> //提供輸入輸出功能，如 cin / cout。
#include <stdio.h> //提供使用 scanf / printf 等等。
```

### 主程式(主函式)
每個C++程式都會有至少一個的函式，且必須包含名為main的主程式。
```cpp=
int main(){
    //程式內容
}
```

### 命名空間
命名空間就像是一個識別編號或分類櫃子但實際上，std 幾乎包含了所有常用 C++ 標準庫功能。
當程式中有很多功能或來自不同來源的函式、變數，可能會不小心取到相同的名字，這時就需要命名空間來幫忙分類與區隔。

```cpp=
#include <iostream>
using namespace std; //一般習慣加在標頭檔後
```
如果沒有加入這一行,後面的每一個`cin` `cout`都要寫成`std::cin` 跟 `std::cout`

所以一開始的例子如果加入命名空間,也可以寫成
```cpp=
#include <iostream>
using namespace std;
int main(){
    cout << "Hello, world" << endl; //cout前面不用加上std:: 因為第二行加入了命名空間std
    return 0;
}
```

### 輸入輸出
輸入:`cin`
輸出:`cout`
他們是被定義在 `std // 命名空間` 、`iostream// `函式庫之下的，所以必須要引入該函式空間和命名空間才能使用。
再看回剛剛的程式碼
```cpp=
#include <iostream>
using namespace std;
int main(){
    cout << "Hello, world" << endl; 
    cout << 'c';
    return 0;
}
```
使用`" "`來包字串，`' '`用來包字元
而後面的`endl`又是幹嘛的呢?
它會讓你的輸出換行，所以程式輸出結果為
```css
Hello, world
c
```
### 回傳值
`return 0;` 可以理解為回傳一個值,告訴電腦程式已經執行完畢(return還有其他用法,但那是之後的事情了),在編譯器中如果沒有`return 0;`也會自動結束程式(包含DanDanJudge)，但最好還是養成習慣寫，有些如果沒寫會無法編譯。(本人沒有遇過 也從不寫 但還是呼籲寫一下)
