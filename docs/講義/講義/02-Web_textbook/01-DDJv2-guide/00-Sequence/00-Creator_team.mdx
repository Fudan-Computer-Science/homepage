在正式介紹成員之前 先簡單介紹一下這次 DDJv2 線上評測系統的誕生背景吧！
DDJ 要從 zerojudge (ZJ) 搬到新的平台上 這個想法其實在 11 屆 就已經開始萌芽了  
最一開始是打算用TIOJ來做為新的平台 但後來發現 TIOJ 的系統架構並不適合我們的需求
所以有兩條路走  
1. 修改 TIOJ 的原始碼來符合我們的需求
2. 使用其他現有的評測系統
原計畫是走第一條路 但後來發現 TIOJ 的原始碼實在是太過複雜了  

於是 14 屆幹部決定走第二條路
14 屆幹部剛開始的時候是打算用`青島OJ`來做為新的平台
但在實際測試的過程中 發現 青島OJ 的系統架構也不太適合我們的需求
於是最後找到了用`HydroOJ`來作為新的平台
關於 HydroOJ 的介紹可以參考序章的最後一部份或接下來的 DDJv2 系統介紹

好了 接下來就正式介紹這次 DDJv2 線上評測系統的創作團隊成員吧！

我 草貓 是找到 HydroOJ 的那個人
因此 這次 DDJv2 線上評測系統的架設有大部分都是由我負責的
:::info[?=spoiler為什麼我會選HydroOJ]
最重要的原因是 HydroOJ 的系統架構可以直接使用 npm package 去安裝各式各樣的客製化功能  
套件可以實現各種功能 像部落格等等 自己也可以開發自己的套件來實現想要的功能  
這樣後繼的幹部就可以持續開發新的功能
讓DDJv2 可以一直進步下去 日月光華同燦爛 向前向前向前進展
而且 HydroOJ 原生就支援多種功能(像是多語言支援、功課、比賽氣球等等)  
希望大家會喜歡這個新的平台！  
:::
