## 1.1 為什麼你該使用DDJ_v2
### I. DDJ_v2特色
    |              | DDJ_v2(Hydro)          | DDJ(Zerojudge)           |
|:------------ |:---------------------- |:------------------------ |
| 定位         | 高度客製化             | 入門到基礎競賽           |
| 架設門檻     | 中低                   | 極低                     |
| 自行維護成本 | 極低                   | 幾乎不可自行維護         |
| 題目來源     | 具有一定程式能力的幹部 | 具有程式設計能力的任何人 |
| 題單         | 多且完整               | 少，且資源零散難以整理   |
| 題解與討論區 | 有                     | 永久關閉                 |
| 支援語言     | C/C++/Python等         | C/C++/Python/Java        |

自2025/2/23 起，DDJ_v1不再提供維護，將來可能會關閉註冊功能。
### II. 神秘結界--域
域(domain)是一個獨立的空間，讓不同房間的題目不會互相混淆。我們在DDJ_v2上開了兩個域:

1. [復旦程式設計班](https://dandanjudge-v2.fdhs.tyc.edu.tw/d/system/domain/join)
    這個題庫以教學和競賽培訓為基礎，打造一個不論是C++新手還是練習多年的電神都能練習的地方。
    此域所有人都能加入，但復旦程式設計班的應屆學員會有專屬的班級功能。
    若想出題，則必須為程式設計班的應屆/歷屆幹部。

2. [旦旦解題農場附設試驗所](https://dandanjudge-v2.fdhs.tyc.edu.tw/d/code_lab/join)
    這個題庫可以讓所有人出題，不論是想出好題，或者出一些奇怪的題目、辦一些奇妙的比賽，只要登入即可開啟所有權限。
    幹部們會定期去試驗所找一些好題納入復旦程式設計班的域!

## 1.2 如何註冊DDJ_v2?
1. 進入 [復旦程式設計班](https://dandanjudge-v2.fdhs.tyc.edu.tw/d/system/domain/join)
2. 點擊上方橫幅「註冊」，輸入你的電子郵件
3. 到你的電子信箱，找到一封fdhscpp寄送的Sign Up郵件，點擊裡面的連結，並依照指示完成註冊手續。
4. 如果你想加入我們的另一個域，請點擊[旦旦解題農場附設試驗所](https://dandanjudge-v2.fdhs.tyc.edu.tw/d/code_lab/join)。

## 1.3 關於權限
對於[復旦程式設計班](https://dandanjudge-v2.fdhs.tyc.edu.tw/d/system/domain/join)來說，我們將權限分成四種等級:
lv1 未登入者: 僅可檢視，不可提交或修改。

lv2 一般使用者: 可以提交題目、發題解、參與討論或比賽，但是不可出題。

lv3 出題者: 程式設計班與資訊研究社幹部可持有此權限，可以出題但是不能刪除其他人的題目。

LV4 管理者: 僅[協助DDJ_v2架設者](https://fudan-computer-science.github.io/homepage/docs/%E8%AC%9B%E7%BE%A9/%E8%AC%9B%E7%BE%A9/Web_textbook/DDJv2-guide/Sequence/Creator_team/)
及程式設計班14屆以後的各屆負責人可持有。


對於[旦旦解題農場附設試驗所](https://dandanjudge-v2.fdhs.tyc.edu.tw/d/code_lab/join)來說，權限僅分成三種:
未登入者、出題者、管理者。登入後即可獲得出題者權限。
