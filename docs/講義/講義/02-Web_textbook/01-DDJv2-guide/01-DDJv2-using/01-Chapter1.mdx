## 1.1 為什麼你該使用DDJ_v2
### I. DDJ_v2特色
    |              | DDJ_v2(Hydro)          | DDJ(Zerojudge)           |
|:------------ |:---------------------- |:------------------------ |
| 定位         | 高度客製化             | 入門到基礎競賽           |
| 架設門檻     | 中低                   | 極低                     |
| 自行維護成本 | 極低                   | 幾乎不可自行維護         |
| 題目來源     | 具有一定程式能力的幹部 | 具有程式設計能力的任何人 |
| 題單         | 多且完整               | 少，且資源零散難以整理   |
| 題解與討論區 | 有                     | 永久關閉                 |
| 支援語言     | C/C++/Python等         | C/C++/Python/Java        |

自2025/2/23 起，DDJ_v1不再提供維護，並將於不久後關閉註冊功能。
### II. 神秘結界--域
域是一個獨立的空間，讓不同房間的題目不會互相混淆。我們在DDJ_v2上開了兩個域:

1. [復旦程式設計班](https://dandanjudge-v2.fdhs.tyc.edu.tw/d/system/domain/join)
    這個題庫以教學和競賽培訓為基礎，打造一個不論是C++新手還是練習多年的電神都能練習的地方。
    此域所有人都能加入，但復旦程式設計班的應屆學員會有專屬的小組功能。
