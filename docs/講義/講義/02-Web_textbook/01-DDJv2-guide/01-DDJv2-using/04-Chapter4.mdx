## 1.1 導入題目
目前已經完全支援從舊版DDJ及其他基於Zerojudge的解題網站搬運題目。

### I. 使用方式
1. 首先，你要先有該題目的擁有權
2. 在舊版DDJ上點選匯出題目，之後你會得到一個zjson檔案。如果這個檔案大於100MB，請先把zjson檔案壓縮為.zip。
3. 到DDJ_v2上傳題目的地方，點選上傳並選取你的檔案，會自動幫你匯入到DDJ_v2並完成調整格式、測資上傳等作業。
