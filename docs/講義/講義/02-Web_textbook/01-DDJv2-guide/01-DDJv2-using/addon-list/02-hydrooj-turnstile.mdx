# cloudflare-turnstile
Cloudflare Turnstile 訪客認證系統 提供在網站後台設定任意表單啟用 Turnstile 的功能
安裝此套件後可以在網站後台的[「配置管理」](https://dandanjudge-v2.fdhs.tyc.edu.tw/manage/addons)中看到
可以新增需要的頁面的 data-page 和表單按鈕的 Xpath 來新增 Turnstile

[npm package 連結](https://www.npmjs.com/package/hydrooj-turnstile)
