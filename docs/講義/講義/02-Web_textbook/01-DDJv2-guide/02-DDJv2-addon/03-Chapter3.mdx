## 1.1 Locale 資料夾介紹
在HydroOJ套件中 如果需要建立多國語系 就需要建立`locale`資料夾
```
my-addon/
├── locale/
    ├── zh_TW.yml
```
其檔名會是語言代碼加上`.yml`副檔名 (如`zh_TW.yml`代表繁體中文)
支援的語言代碼如下表所示
| 語言 | 語言代碼 |
| ---- | -------- |
| 繁體中文 | zh_TW |
| 簡體中文 | zh |
| 英文 | en |
| 韓文 | ko |
若需支援其他語言 可對 hydroOJ repo [PR](https://github.com/hydro-dev/Hydro/pulls)

### I. yml檔案結構
```yml
some key: 這是一段文字
another key: 這是另一段文字
```
可以藉由`key`來定義不同的文字內容
`key`通常會和`en.yml`中的value相同

### II. 使用多國語系

#### 在 Handler 中使用多國語系

```typescript
import { Handler, Types } from 'hydrooj';

class MyHandler extends Handler {
    async get() {
        // 獲取當前語言的翻譯
        const title = this.ctx.i18n.load(this.user.preferredLanguage).gettext('my_page_title');
        this.response.body = { title };
    }
}
```

#### 在 index.ts 中載入多國語系

```typescript
import { Context } from 'hydrooj';

export async function apply(ctx: Context) {
    // 載入中文翻譯
    ctx.i18n.load('zh_TW', {
        'my_page_title': '我的頁面',
        'welcome_message': '歡迎使用！',
    });
    
    // 載入英文翻譯
    ctx.i18n.load('en', {
        'my_page_title': 'My Page',
        'welcome_message': 'Welcome!',
    });
}
```

#### 前端中使用多國語系

```jsx
import { i18n } from '@hydrooj/ui-default';

// 獲取當前語言的翻譯
const title = i18n('my_page_title');
console.log(title);  // 根據用戶語言顯示相應翻譯
```

## 1.2 Templates 資料夾介紹
在HydroOJ套件中 如果需要渲染 HTML 就可以建立`templates`資料夾方便渲染

### I. 模板檔案結構與基礎
其採`nunjucks`語法來撰寫 HTML 
使用時將傳入參數 而後渲染成最終的 HTML
```html
{% extends "layout/basic.html" %}
{% block content %}
<h1>{{ title }}</h1>
<p>{{ message }}</p>
{% endblock %}
```
其中
- `{% extends "layout/basic.html" %}`：代表繼承自 HydroOJ 預設的 `basic.html` 模板
- `{% block content %} ... {% endblock %}`：代表這個區塊會被注入到 `basic.html` 模板中的 `content` 區塊 *需和預設模板中的區塊名稱相同*
- `{{ title }}` 和 `{{ message }}`：代表會被替換成傳入的參數值
可以在 HydroOJ repo 中查看[預設模板](https://github.com/hydro-dev/Hydro/tree/master/packages/ui-default/templates)

若想要覆蓋其他模板 可手動複製貼上到自己的`templates`資料夾中來覆蓋原有的模板

### II. 多國語系在模板中的使用

可以通過以下方式在模板中使用多國語系文字
```html
{{ _('the key') }}             <!-- 簡單翻譯 -->
{{ _('greeting', { name: name }) }} <!-- 帶參數的翻譯 -->
```

### III. 高級 Nunjucks 語法

**條件判斷**
```html
{% if success and result %}
<script>
    const message = {{ result | dump | safe }};
    alert('✓ success:  ' + message);
</script>
{% elif error %}
<p class="error">{{ error }}</p>
{% else %}
<p>請稍候</p>
{% endif %}
```

**迴圈**
```html
{% for package in packages %}
<li>{{ package.name }} - {{ package.version }}</li>
{% endfor %}

<!-- 帶索引的迴圈 -->
{% for item in items %}
<div>第 {{ loop.index }} 項: {{ item }}</div>
{% endfor %}
```

**過濾器**
```html
{{ value | upper }}           <!-- 轉換為大寫 -->
{{ value | lower }}           <!-- 轉換為小寫 -->
{{ value | abs }}             <!-- 絕對值 -->
{{ array | length }}          <!-- 陣列長度 -->
{{ object | dump }}           <!-- 轉換為字串 -->
{{ html_string | safe }}      <!-- 標記為安全的 HTML -->
```

**設定變數**
```html
{% set myVar = 'Hello' %}
<p>{{ myVar }}</p>

{% set myArray = [1, 2, 3] %}
{% for item in myArray %}
  <span>{{ item }}</span>
{% endfor %}
```

**模板中可用的內置變量**
```nunjucks
<!-- 使用者相關 -->
{{ user._id }}           <!-- 當前用戶 ID -->
{{ user.uname }}         <!-- 當前用戶名 -->
{{ user.email }}         <!-- 當前用戶郵箱 -->
{{ user.hasPerm(perm.PERM_VIEW) }} <!-- 權限檢查 -->

<!-- 域相關 -->
{{ domain._id }}         <!-- 當前域 ID -->
{{ domain.name }}        <!-- 域名稱 -->
{{ domain.owner }}       <!-- 域擁有者 -->

<!-- 頁面相關 -->
{{ page_name }}          <!-- 當前頁面名稱 -->
{{ UiContext.cdn_prefix }} <!-- CDN 前綴 -->

<!-- 常數 -->
{{ perm.PERM_VIEW }}     <!-- 權限常數 -->
{{ PRIV.PRIV_EDIT_SYSTEM }} <!-- 特權常數 -->

<!-- 翻譯函數 -->
{{ _('translation_key') }} <!-- 翻譯函數 -->
```

### IV. 在 Handler 中使用模板

```typescript
import { Handler } from 'hydrooj';

class MyPageHandler extends Handler {
    async get() {
        // 指定要使用的模板
        this.response.template = 'my_page.html';
        
        // 傳入參數 會被注入到模板中
        this.response.body = {
            title: '我的頁面',
            message: '歡迎訪問！',
            items: ['項目1', '項目2', '項目3'],
            success: true
        };
    }
}

// 然後在 my_page.html 模板中使用這些參數
```
