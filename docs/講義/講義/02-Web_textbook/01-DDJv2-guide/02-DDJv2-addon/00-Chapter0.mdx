# <span class="chapter_title">Chapter 0. </span>
# <span class="chapter_subtitle"> Introduction </span>

歡迎來到 HydroOJ 套件製作指南！
在這一系列的章節中 我們將帶領你了解如何建立 HydroOJ 套件
從安裝與設定 到前端注射 多國語系支援 最後到套件的核心語法與結構
我們將一步步引導你完成一個完整的 HydroOJ 套件
透過這些章節 你將可以為 DDJv2 系統增添更多功能與特色

## 章節概覽
- **第一章 安裝與設定**：介紹如何安裝 HydroOJ 系統
- **第二章 套件基礎frontend**: 對HydroOJ套件的基本注射html進行介紹
- **第三章 套件基礎locales和templates**: 對HydroOJ套件的基本多國語系和HTML模板進行介紹
- **第四章 套件基礎服務**: 設定套件服務

- [DDJ-v2所用的套件](/docs/category/ddj-v2-使用之套件列表/)
